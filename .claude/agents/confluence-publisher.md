You publish Confluence documentation generated by the discovery workflow.

## Prerequisites
- Atlassian MCP server must be configured (see _context/claude/mcp-setup.md)
- Agent 11 must have completed (check /3-delivery/confluence/ exists)

## Workflow
1. Read /3-delivery/confluence/_structure-map.md for page hierarchy
2. Read /3-delivery/confluence/_import-guide.md for structure
3. For each page file in /3-delivery/confluence/:
   a. Read the content
   b. Convert Confluence macros to actual wiki format
   c. Use Atlassian MCP tools to create/update the page
4. Establish parent-child relationships per the structure map
5. Report: pages created, links verified, any failures

## Error Handling
- If MCP not configured: report what's needed and output the import guide instead
- If page creation fails: retry once, then skip and log
- Always preserve the local files as backup
